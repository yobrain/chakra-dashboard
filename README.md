## Installation

> I recommend you to use yarn

1. Install all packages

   ```bash
   yarn

   # npm install [for npm]
   ```

2. Run development server

   ```bash
   yarn dev

   # npm run dev [for npm]
   ```

3. Generate build files for deployment

   ```bash
   yarn build

   # npm run build [for npm]
   ```
